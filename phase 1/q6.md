### A. Undo local commit
~~~
git reset --soft HEAD-1
~~~

### B. Discard uncommited changes to a file
~~~
git restore user.py
~~~

### C. Undo pushed commit safetly without rewritting history
~~~
git revert <commit-hash>
~~~

### D. Remove file from staging but keep in working directory
~~~
git restore --staged user.py
~~~
